### 1. 결과 페이지 모바일 헤더 레이아웃 개선
**파일**: `app/(main)/result/page.tsx`

**문제**:
- 모바일에서 헤더 텍스트("생성 결과")가 버튼들과 같은 줄에 있어 공간이 좁아짐
- "1장의 이미지가 생성되었습니다 • 5 크레딧 사용" 텍스트가 심하게 줄바꿈됨

**해결**:
- 헤더와 버튼을 수직 배치로 변경 (`flex-col`)
- 모바일에서 버튼 아이콘만 표시 (`hidden sm:inline`)
- 데스크탑에서는 아이콘 + 텍스트 표시
- 버튼 사이즈 `size="sm"`으로 축소
- `flex-wrap` 적용으로 공간 부족 시 자동 줄바꿈
