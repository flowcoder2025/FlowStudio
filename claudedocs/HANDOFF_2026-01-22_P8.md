## 구현 세부 내용

### 1. GuideChat.tsx
- 대화형 채팅 UI로 가이드 단계 진행
- 옵션 선택 및 텍스트 입력 지원
- 자동 스크롤, 로딩 상태, 리셋 기능
- `DynamicGuide` 타입과 연동

### 2. RecommendCard.tsx
- 3가지 variant: `default`, `compact`, `featured`
- 스코어 배지, 태그 표시
- `WorkflowRecommendation` 타입 사용
- 추가 컴포넌트: `RecommendList` (리스트 뷰)

### 3. StepFlow.tsx
- vertical/horizontal orientation 지원
- 단계 상태: completed, current, upcoming, skipped, locked
- 진행률 표시, 스킵 버튼
- 추가 컴포넌트: `MiniStepIndicator` (미니 진행 표시)

### 4. ImageUpload.tsx
- 드래그 앤 드롭 업로드
- 파일 유효성 검사 (형식, 크기)
- 업로드 진행률 표시
- 미리보기, 확대 보기, 삭제
- 추가 컴포넌트: `ImageThumbnailList` (읽기 전용 썸네일)

### 5. SimilarWorkflows.tsx
- 크로스 인더스트리 추천 강조
- 가로 스크롤 카드 뷰
- 추가 컴포넌트: `CrossIndustryList`, `IndustryNavigation`
